A framework is a way of buying goods and services. Each framework has its own list of suppliers, and each supplier on the framework has been assessed for compliance with procurement regulations, ease of use, suitability and value for money. Frameworks also have guidance on how to buy.

The Department for Education has a number of recommended frameworks that schools can use. You can see a list of catering frameworks recommended by DfE in the Find a framework tool at https://buy-for-your-school-staging.herokuapp.com/beta/phase-5/find-a-framework/matching-frameworks.

There are lots of advantages to using frameworks. Using a framework:

  • reduces how much time it takes to do your procurement
  • ensures compliance with procurement law (Public Contract Regulations 2015)
  • contract terms are already in place

While there are lots of benefits to using a framework, it might not be right for you in some situations:

  • only suppliers that are listed on a framework can be invited to bid, however, in certain circumstances, suppliers can be added
  • you may not be able to make significant changes to the contract terms

If you haven’t used frameworks before, there’s some useful guidance on the GOV.UK site about frameworks and how to use them at https://buy-for-your-school-staging.herokuapp.com/beta/phase-5/gov/find-the-right-way-to-buy
